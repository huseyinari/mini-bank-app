# Hüseyin Arı Bank Application

Kullanıcı kayıt-giriş işlemleri, hesap oluşturma-güncelleme-listeleme, hesap hareketleri takibi ve hesaplar arası para transferlerini içeren küçük bir banka uygulamasıdır.



## Uygulamayı Başlatma

```bash
docker-compose up -d
```
